# HIP

## Installation
HIP is available over a CDN
```html
<script src=""></script>
```

If you serve HIP from CDN, we recommend keeping the **crossorigin** attribute set:
```html
<script crossorigin src="..."></script>
```

## Usage
Include the hip-element in your code.
```html
  <hip-element></hip-element>
```
